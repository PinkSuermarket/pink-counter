# Pink Supermarket — Billing Counter Offer Lookup

Internal tool for billing counter staff to look up and redeem WhatsApp marketing offers.

## How to use
1. Open the page URL on any browser at the billing counter
2. Enter the 4-digit staff PIN
3. Type customer's 10-digit phone number → Search
4. Verify the offer and click "Mark as Redeemed"

## Default PIN
`1234` — change this in `index.html` line with `const CORRECT_PIN`
